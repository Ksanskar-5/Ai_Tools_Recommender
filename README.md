# AI Tools Recommender App

This repository contains a standalone Flutter app for discovering AI tools.

## App overview

- Built with Flutter
- Runs without a backend
- Uses bundled local data for search
- Includes Android and web targets

## Run locally

```bash
cd ai_tool_recommender_app
flutter pub get
flutter run -d chrome
```

## Build APK

```bash
cd ai_tool_recommender_app
flutter build apk
```
